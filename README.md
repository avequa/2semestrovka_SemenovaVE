# DevTracker

Веб-приложение для учёта личных проектов и задач разработчика

## Реализовано

- Регистрация и авторизация (сессии Flask)
- CRUD проектов с тегами
- Задачи с дедлайнами: создание, закрытие, переоткрытие
- Дашборд: процент выполнения, количество просроченных задач, статус-прогноз

## Стек

- Python
- Flask (Jinja2, сессии)
- PostgreSQL + psycopg2 (чистый SQL, без ORM)

## Как поднять приложение

1. Создать и активировать виртуальное окружение:

    `python -m venv venv`
    `venv\Scripts\activate`

2. Установить зависимости:

    `pip install -r requirements.txt`

3. В файле `db.py` поменять переменные параметров подключения:

    `dbname=''`
    `user=''`
    `password=''`
    `host=''`
    `port=`

## Запуск

    `python app.py`

Приложение доступно на http://127.0.0.1:5000
